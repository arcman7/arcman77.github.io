JavaScript


JavaScript is an all purpose scripting language with medium-weight syntax, because of this it is well suited for experimentation as well as large scale projects. It has a large base of external libraries to pull from including jQuery, three.js, underscore, and lodash. The libraries lodash and underscore make it's it almost as great a scripting language as ruby by providing many pre-built functions that mirror many of the enumerables and methods availible in ruby.
