Week 4: Enumerable Method: cycle

Skipper's log 3/20/2015:

Hey there, today's tech post will cover a particular enumerable method called cycle. It's a very simple yet useful method that actually allows you to in-line script iteration over an array as many times as you like. With this method for-loops, which I will always love and hold in special place, have virtually no reason to be in your code. This method is easy to call with remarkably very little syntax. Let's just start by recalling what our code would look like without it and build our way up from the basic for loop. In the code block below, an example is shown with an array of 5 elements that we wish to iterate over and print each element 3 times:

a = [1,2,3,4,5]
for i in 1..3 
   for j in 1..5
           
     puts "printing element #{a[j]}"
           
   end
           
end

Equivalent to:


3.times { a.each {|e| puts "printing element #{e}" }}

Which is equivalent to:


a.cycle(3) { puts "printing element #{e}"}
=> nil

As you can see in the code block above, it's very short and serves well to cut down the syntax required for iteration of any type. What's more is that you can actually call cycle without any argument to loop indefinitely. If cycle completes its specified loops over the array without interuption it will return nil.
